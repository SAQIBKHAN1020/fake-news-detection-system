# Fake News Detection System

## Project Overview

This project detects whether a news article is Fake or Real using Machine Learning and NLP techniques.

## Technologies Used

- Python
- Pandas
- NumPy
- NLTK
- Scikit-Learn
- Streamlit

## Machine Learning Model

TF-IDF Vectorizer is used to convert text into numerical features.

Classification Model:
- Logistic Regression

## Features

- User can enter news text
- System predicts Fake or Real news
- Simple Streamlit interface

## Installation

Clone repository:

git clone your_link


Install libraries:

pip install -r requirements.txt


Run:

streamlit run app.py
